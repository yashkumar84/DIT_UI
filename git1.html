📝 Step 2: First Files Create & Commit

public class BankAccount { private String accountNumber; private String accountHolder; private double balance; public BankAccount(String accountNumber, String accountHolder) { this.accountNumber = accountNumber; this.accountHolder = accountHolder; this.balance = 0.0; } public void deposit(double amount) { balance += amount; System.out.println("Deposited: " + amount); } }
public class Transaction { private String transactionId; private double amount; private String type; public Transaction(String transactionId, double amount, String type) { this.transactionId = transactionId; this.amount = amount; this.type = type; } }
public class BankingApp { public static void main(String[] args) { System.out.println("Banking System v1.0"); BankAccount account = new BankAccount("ACC001", "Rahul"); account.deposit(5000); } }

🎯 Ab Git ko batao ki files track karni hain

Terminal
git status
Untracked files: (use "git add <file>..." to include in what will be committed) BankAccount.java Transaction.java BankingApp.java
git add .
git status
Changes to be committed: (use "git restore --staged <file>..." to unstage) new file: BankAccount.java new file: Transaction.java new file: BankingApp.java
git commit -m "Initial commit: Added basic banking classes"
✅ Success! [main (root-commit) a1b2c3d] Initial commit: Added basic banking classes
3 files changed, 45 insertions(+)

💡 Step-by-Step Explanation

1️⃣ git status - Kya kya change hua? Untracked files dikha dega
2️⃣ git add . - Saari files ko staging area mein daal do (. ka matlab = all files)
3️⃣ git status - Confirm karo ki files staged hain
4️⃣ git commit -m "message" - Permanently save with meaningful message!
